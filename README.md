# Subscription Page

This project is a simple and modern **Subscription Page** created using **HTML and CSS**. It displays different subscription plans with clear pricing, features, and a user-friendly layout. The design focuses on clean structure, proper alignment, and responsive styling to provide a professional look.

## Features

* Clean and attractive subscription layout
* Multiple subscription plans with pricing and features
* Well-structured content using HTML
* Styled using CSS for modern UI design
* Responsive and beginner-friendly design
* Proper alignment using Flexbox

## Technologies Used

* HTML5
* CSS3

## Project Purpose

The purpose of this project is to practice frontend development skills, especially:

* Creating structured layouts using HTML
* Designing modern UI using CSS
* Using Flexbox for alignment
* Building real-world components like pricing/subscription sections



## Future Improvements

* Add hover effects on subscription cards
* Add buttons with interactive effects
* Improve responsiveness for mobile devices
* Add JavaScript for plan selection
